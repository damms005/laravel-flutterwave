This repo is a copy (not fork) or [Kingflamez's Flutterwave package](https://github.com/kingflamez/laravelrave) that I had to make simply because the upstream does not always upgrade for new Laravel version compatibility in good time.


## Recommendation

Use [https://github.com/damms005/laravel-multipay](https://github.com/damms005/laravel-multipay) for better experience. It depends on this repo as a [VCS repo](https://getcomposer.org/doc/05-repositories.md#vcs) and is well maintained, receives regular updates, and provides interface for other payment providers.
